# Console example for FeatBit Node.js Server-Side SDK

## Introduction
The current example APP contains two files in `src` folder:
- `esm.ts`: example for using ES module and typescript.
- `cjs.js`: example for using commonjs.


## Get started

```bash
npm install
```

### ES module

```bash
npm run esm
```

### CommonJs

```bash
npm run cjs
```
